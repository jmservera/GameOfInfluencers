How to run the local app

You will need:
  • Azure Storage Emulator 4.4 or greater.Download Here
  • Run Storage Emulator with: 
    C:\Program Files (x86)\Microsoft SDKs\Azure\Storage Emulator>AzureStorageEmulator.exe start
  • Share Sql Server LocalDB instance: 
    sqllocaldb share mssqllocaldb goflocal
  • Add [NETWORK SERVICE] user as dbo to the localdb instance and db in the logins folder (from SSMS)
